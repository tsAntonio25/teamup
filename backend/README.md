# Backend

Backend Folder of the Project  

## Dependencies used

* Express

* Cors

* Mongoose

* Express Async Handler

* Body-Parser

* JWT (Json Web Token)

* Axios

* Bcrypt

To install the following dependencies, run the following command inside the folder  

```bash
npm i express cors mongoose body-parser express-async-handler jsonwebtoken axios bcrypt
```

To run the server, run the following command:

```bash  
node server.js
```
